# Gestión del Proyecto - KriPass

Para el desarrollo de este gestor de contraseñas, estoy aplicando una metodología **Agile/Kanban**.

### Metodología
- **Tablero Kanban**: Utilizo Trello para visualizar el flujo de trabajo.
- **Columnas del tablero**:
  - `To Do`: Tareas pendientes de las fases de la guía.
  - `In Progress`: Tareas que estoy codificando actualmente con ayuda de la IA.
  - `Done`: Tareas verificadas y con commit realizado.

### Organización del Repositorio
- **Monorepo**: El proyecto contiene tanto el `frontend` (React) como el `server` (Node.js) para mantener la coherencia del sistema.
- **Commits**: Se realizan commits descriptivos al finalizar cada hito importante de la guía.
