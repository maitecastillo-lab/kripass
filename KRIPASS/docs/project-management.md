# Gestión del Proyecto - KRIPASS

Para el desarrollo de este gestor de contraseñas, apliqué una metodología Agile/Kanban porque al ser un proyecto individual, me parecía lo más práctico. No tenía sentido usar Scrum con sprints y reuniones si solo era yo.

## Metodología

- **Tablero Kanban:** utilicé Trello para visualizar el flujo de trabajo y tener controlado lo que me quedaba por hacer.

## Cómo organicé las tarjetas

Cada funcionalidad grande la convertí en una tarjeta (por ejemplo, "Login", "Carpetas", "Encriptación AES", "Despliegue"). Luego dentro de cada tarjeta iba anotando las subtareas más pequeñas que implicaba, tipo:

- Crear el componente
- Conectarlo con el backend
- Probarlo con varios casos
- Arreglar los bugs que fueran saliendo

Esto me ayudó a que las tareas no fueran tan abrumadoras y a ir avanzando por partes.

## Cómo lo usé durante el desarrollo
