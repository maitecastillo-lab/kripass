# 🛡️ KriPass - Gestor de Contraseñas

Bienvenido a **KriPass**, un sistema seguro para la gestión y generación de contraseñas, desarrollado con una arquitectura moderna de software.


##Gestión del Proyecto
El desarrollo de este proyecto se gestiona mediante metodología Kanban.

##Arquitectura del Sistema
El proyecto utiliza una estructura de Monorepo para mantener el Frontend y el Backend sincronizados:

###Frontend (React)
- **Framework**: Vite + React + TypeScript.
- **Estilos**: Tailwind CSS v4.
- **Organización**: Estructura de carpetas por componentes, páginas y hooks.

### Backend (Node.js)
- **Motor**: Node.js + Express.
- **Arquitectura**: Diseño por capas (Routes, Controllers, Services).
- **Seguridad**: Variables de entorno (`.env`) y validación en frontera de red.
